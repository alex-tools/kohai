# kohai - pluggable irc bot for managing real-time data events

*I am Kohai, semi-useful communications-facilitating pseudointelligence!*

##v0.0.1 - Experimental

Due to the high demand of requests in the #Node.js and #nodejitsu IRC rooms on irc.freenode.net, we've open-sourced this library. 

kohai is **EXPERIMENTAL SOFTWARE** and has **NO UNIT TESTS**.


At Nodejitsu we try to maintain a high level of quality across all of our open-source projects. This particularly project is not suitable for public use yet. We've released it early because we've gotten a lot of requests from eager developers who want to start hacking in more features.

We look forward to seeing your patches and continuing to improve this library.

# Installation

     git clone git://github.com/nodejitsu/kohai.git
     cd kohai
     node bin/server
     

kohai will now start up and connect to it's default channels on irc.freenode.net.
